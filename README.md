# Java String Tasks

This project contains solutions to 12 different Java String exercises.  


## What This Program Does

The program demonstrates how to:

- Get the length of a string.
- Find a character at a specific index.
- Search for a character inside a string.
- Extract substrings.
- Change text to lowercase and uppercase.
- Replacing the Words.
- Trim whitespace.
- Convert numbers to strings.
- Split strings into parts.
- Convert strings to char arrays.
- Convert char arrays back into strings.

Each task prints its result to the console when the program runs.
